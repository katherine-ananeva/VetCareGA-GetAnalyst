# VetCareGA–GetAnalyst
VetCareGA is an application for managing veterinary clinic services. It helps users register pets, book vet appointments, view visit details, manage pet profiles, and track upcoming care. 
The project focuses on clear REST API design, user-friendly workflows, and secure access for authorized users.
